# Tetris Project

A classic Tetris clone built in Python.

- Prerequisites
* Python 3.x
* Pygame (install via `pip install pygame`)

- How to Run
1.  Clone the repository:
    ```bash
    git clone [https://github.com/swarkive3/Tetris-project.git](https://github.com/swarkive3/Tetris-project.git)
    ```
2.  Navigate to the folder:
    ```bash
    cd Tetris-project
    ```
3.  Run the game:
    ```bash
    python tetdel2.py
    ```

- Controls
* **Arrow Keys**: Move and Rotate blocks
* **Down Key**: Speed up fall
